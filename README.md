# Regulatory Burden Dashboard — Republic of Uzbekistan (2019-2024)

An interactive analytical dashboard analyzing **5,793 draft legal acts** published for public consultation on Uzbekistan's regulation.gov.uz portal.

## 🔗 Live Dashboard

[View the Dashboard](https://kamzay97.github.io/regulatory-dashboard-uzbekistan/)

## 📊 Key Findings

| Metric | Value |
|--------|-------|
| Total Draft Regulations | 5,793 |
| Public Comments | 46,019 |
| Government Bodies | 261 |
| Zero-Comment Rate | 59% |
| Peak Year | 2019 (1,255 drafts) |
| Decline Since Peak | -65% |

## 📈 Dashboard Features

- **Overview**: KPI cards, yearly trends, regulation types distribution
- **Trends**: Year-over-year change analysis, monthly patterns
- **Sectors**: Breakdown across 19 policy areas
- **Institutions**: Top regulatory bodies analysis
- **Participation**: Public engagement metrics and gap analysis

## 🗂️ Data Source

Data extracted from [regulation.gov.uz](https://regulation.gov.uz) — the official unified portal for public discussion of draft regulatory documents of the Republic of Uzbekistan.

## 📅 Coverage

July 2018 — January 2025
